Getting Started
What is jintell?

jintell is an all‑in‑one web library for building application‑focused UI/UX websites. It targets single‑page applications, portals, and desktop‑style web applications. The library does not require developers to modify their existing project structure. jintell is not a framework—and never will be.

Key Features
  • Independent: No-nonsense files or changes to your website or application
  • Serverless: No server‑side code required
  • Keyboard Accessibility: Every feature is fully usable without a mouse
  • Object-oriented: Customizable with Javascript rather than Html
  • Predefined: Customizable with HTML rather than JavaScript
  • Built-in: Built‑in UI controls, demo pages, and more.
How to install jintell?
  • Adds jintellv3 to your project - https://github.com/XHighIntell/jintellv3
<script src="/static/lib/jintellv3/intell.min.js"></script>
