Naming Guidelines
These guidelines help you understand the library and make customization more easily.
Control Names

Pascal Case prefix with X-.

.X-Menu .X-Numeric .X-Portal
State Names

Uppercase for any state name accessed by JavaScript.

.SELECTED .ACTIVE .DISABLED .HAS-CHILDREN .CHILDREN-VISIBLE .CHILDREN-OUT
Generic Names
✓ DO use the child combinator (>) for generic names.
.article>.icon{} .article>.title{}
✗ DO NOT use a generic name by itself in a selector.
.icon{} .title{}
